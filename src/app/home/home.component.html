Usage Examples

Get Characters

fetch('https://api.api-onepiece.com/v2/characters/en')
  .then(response => response.json())
  .then(data => console.log(data))

fetch('https://api.api-onepiece.com/v2/fruits/en')
  .then(response => response.json())
  .then(data => console.log(data))
